### 使用方法
> `CLOUD_USERNAME` : 账号（手机号），多个使用&分隔。比如：19900000000&19900000001&19900000002
> `CLOUD_PASSWORD` : 密码，原密码中不能包含&，否则会被识别为分隔符。 比如：PASSWORD0000&PASSWORD0001&PASSWORD0002
多个账号密码按照顺序分别对应，比如：上面的 `PASSWORD0001` 对应 `19900000001`。
